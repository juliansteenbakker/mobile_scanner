# Security Policy

## Supported Versions

Only the latest released version of `mobile_scanner` is supported with security updates.

## Reporting a Vulnerability

If you discover a security vulnerability, please **do not** open a public GitHub issue. Instead, report it privately using [GitHub's private vulnerability reporting](https://github.com/juliansteenbakker/mobile_scanner/security/advisories/new).

Please include as much detail as possible (affected platform, reproduction steps, potential impact) so the report can be triaged quickly.
